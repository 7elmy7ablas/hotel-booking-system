# Prompts Library

Collection of reusable prompts for AI-assisted development of the hotel booking system.

## Categories
- Feature development prompts
- Code review prompts
- Testing prompts
- Documentation prompts
- Architecture prompts
- Database prompts

## Purpose

This library contains reusable AI prompts designed to accelerate development of the hotel booking system. These prompts help maintain consistency, follow best practices, and reduce repetitive prompt engineering when working with AI assistants.

## Using with GitHub Copilot

1. **Copy the prompt**: Select and copy the desired prompt from this library
2. **Paste as comment**: Add the prompt as a comment in your code file
3. **Let Copilot generate**: Press `Tab` or `Ctrl+Enter` to accept Copilot suggestions
4. **Refine if needed**: Modify the prompt or add context for better results
