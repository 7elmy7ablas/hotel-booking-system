# Hotel Booking System - Project Structure

```
hotel-booking-system/
│
├── sprints/
│   └── README.md
│
├── prompts-library/
│   └── README.md
│
├── architecture/
│   └── README.md
│
└── .github/
    └── ISSUE_TEMPLATE/
        ├── bug_report.md
        └── feature_request.md
```

## Folder Descriptions

### sprints/
Sprint planning documents, user stories, and retrospectives. Each sprint gets its own markdown file with goals and outcomes.

### prompts-library/
Reusable AI prompts for development tasks including feature development, code reviews, testing, and documentation.

### architecture/
Technical documentation including system design, database schemas, API specs, and infrastructure diagrams.

### .github/ISSUE_TEMPLATE/
GitHub issue templates for standardized bug reports and feature requests.
